# Survey on Images and Icons in Catholic Faith

## Overview
This project contains data collected from a survey conducted with Catholics, focusing on their perceptions of religious images, icons, and the use of artificial intelligence in generating sacred representations.

## Data File
- **File name:** `survey_catholics_images.csv`
- **Format:** CSV (Comma-Separated Values)
- **Encoding:** UTF-8 (compatible with Portuguese special characters)

## Data Structure
The file contains 24 responses, each representing one participant. The columns include:

### Metadata Columns
- `No.`: Sequential response number
- `Response ID`: Unique response identifier
- `Timestamp`: Date and time of submission (format: YYYY-MM-DD HH:MM:SS)
- `Finished`: Survey completion status ("Yes"/"No")
- `Survey ID`: Survey identifier
- `Formbricks ID (internal)`: Internal Formbricks platform ID
- `User ID`: User identifier (empty in this dataset)
- `Notes`: Notes (empty)
- `Tags`: Tags (empty)
- `url`: Survey URL
- `country`: Respondent's country (all "BR" - Brazil)
- `userAgent - os`: Device operating system
- `userAgent - device`: Device type
- `userAgent - browser`: Browser used

### Survey Questions
1. **Idade / Age** – Respondent's age
2. **Acreditas que algumas imagens e ícones devem ser perfeitas para agradar a Deus? / Do you believe that some images and icons must be perfect in order to please God?** – Responses: "Sim / Yes", "Não / No", "Não sei / I don't know"
3. **Tu evitas reverências às imagens de Jesus Cristo, da Santíssima Virgem e demais santos se essas imagens são geradas por inteligência artificial? / Do you avoid revering images of Jesus Christ, the Blessed Virgin and other saints if these images are generated by artificial intelligence?** – Responses: "Sim / Yes", "Não / No", "Depende / It depends"
4. **Posso, posteriormente, entrar em contato mediante necessidade? / Can I contact you later if I need to?** – Responses: "Sim / Yes", "Não / No"
5. **Contato / Contact** – Respondent's name or identifier
